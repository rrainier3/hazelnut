Cre-animate.js - Onscroll Animations

Introduction

Cre-animate.js was created to be an incredibly simple way to add onscroll animations to any of your websites elements. This includes text, images, videos any html tag you want can be anmated into view.

You have full control over the animation type, speed, delay, scroll offset and easing effect.

  • Animation Type refers to how the element will animate into view. (Example: Fade-In, Slide-Up)
    There are 11 options in total for animation types. More on this later.
     
  • Speed is the time it will take for the animation to take place in milliseconds. 1000ms is equal to 1 second.
     
  • Delay is particularily useful when animating multiple elements on the same line. You can have all the elements animations take place at the same trigger point, but they can all have different delay times in milliseconds which will allow them to animate into view at different times.
     
  • Scroll Offset is the "trigger point" or point on the screen in which the animation will start to take place. A offset value of "0" means the animation will take place when the element hits the top of the viewport(screen). A value of "100" means the element will animate into view as it enters the viewport(screen). Scroll offset can have a value of anything between 0-100.
     
  • Easing Effect is the way the element will animate into view from start to finish. (Example: An easing effect of "Linear" means the element will animate into view at the same speed from start to finish. An easing effect of "EaseIn" means the element will start animating slowly and end quickly.) There are 27 easing effects to have fun with.

Implementation

Cre-animate.js was created to be incredibly easy to implement.

You must include 2 files on your webpage.
jQuery:
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
Cre-animate.js:
<script type="text/javascript" src = "js/jquery.cre-animate.js"></script>
Remember to ensure you have the correct path to the jquery.cre-animate.js file.

Once you include both of these files on your page, you are ready to start animating!

Adding Animations

Adding animations to elements is very simple.

There are two ways you can add animations to elements.

1. Use the Animation Generator included. cre-animation-generator.html

This Generator will allow you to select the options for your animations and generate a wrapper code.
The wrapper code is a <div> tag that you will place the elements code inside of that you want animated.
Example: <div class="cre-animate" animation-options>Your text or image or any html content goes here</div>

It's that simple. Where you see "animation-options" there will be multiple data-attribute tags for your specified options.

Re-load your page and scroll to the element to watch the onscroll animation take place!

 

2. Advanced Options - Manually generate the animation code.

Advanced

If you would like to get down and generate the animation code yourself this section will explain exactly how the script works.

There are two ways you can add the required animation information to an element.

1. Create a wrapper <div> tag around the element you wish to animate. This is what the Animation Generator does.
Add the class="cre-animate" to the div. The script searches for elements with this class and runs the script to animate.

or

2. You can add the class="cre-animate" directly to the element you wish to animate. This sometimes causes unexpected results as the cre-animate class has a position set to relative.

 

Once you create the wrapper <div class="cre-animate"> or add the class="cre-animate to your element the following data-attributes can be applied.
 

Data-Animation:
Add data-animation="animation-type" to the wrapper class.
"animation-type" value can be any of the following:
fade-in
fade-out
slide-down-from-top
slide-in-from-right
slide-up-from-bottom
slide-in-from-left
scale-up
scale-down
rotate
flip-y-axis
flip-x-axis
 

Data-Speed:
Add data-speed="speed" to the wrapper class.
"speed" value can be any numerical value for how quickly the animation will take place. The speed attribute is in milliseconds. For example a value of "1000" will equal 1 second.
 

Data-Delay:
Add data-delay="delay time" to the wrapper class.
"delay-time" value can be any numerical value for delaying the animation. This attribute is again in milliseconds.


Data-Offset:
Add data-offset="trigger-point" to the wrapper class.
"trigger-point" value can be any number from 0-100 which indicates the trigger point in percent where the animation will take place. 0% = top of viewport, 100% = bottom of viewport, 50% = middle of the screen.
The element will not animate until the top of the element reaches the trigger point specified.


Data-Easing:
Add data-easing="easing value" to the wrapper class.
"easing value" can be any of the following 27 easing options:
ease
in
out
in-out
snap
easeOutCubic
easeInOutCubic
easeOutCirc
easeInOutCirc
easeInExpo
easeOutExpo
easeInOutExpo
easeInQuad
easeOutQuad
easeInOutQuad
easeInQuart
easeOutQuart
easeInOutQuart
easeInQuint
easeOutQuint
easeInOutQuint
easeInSine
easeOutSine
easeInOutSine
easeInBack
easeOutBack
easeInOutBack

When you're complete your code should look something like this:
<div class="cre-animate" data-animation="fade-in" data-speed="1000" data-delay="500" data-offset="50%" data-easing="easeInOutBack">Your Object or Element to be animated goes here</div>
